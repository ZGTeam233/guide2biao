#### 安装安卓（虚拟机）
1.  应用商城下载 **VMOS Pro**。
2.  打开，选择手动。
3.  新建云虚拟机，上方切到*本地虚拟机*，下载 **android 9 尝鲜版ROM** ，*不要下载推荐镜像*，下载好后运行。

#### 安装linux（proot容器）
1.  将下载好的APK文件放入虚拟机安装。
2.  打开，自动安装，直到出现XFCE4桌面，主题类似于windows桌面。

#### 安装WINE Is Not an Eumlated
