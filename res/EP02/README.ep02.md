#### 开始之前
-   翻看路由器上的标签，一般在其背面/底面，找到管理地址/账号/密码，形如
```
品名：无线路由器
型号：RB03
输入：12伏1.5安
管理后台：http://192.168.31.1
管理后台账号：root
管理后台密码：password
```
-   用浏览器以*网址*形式访问管理地址，输入账号/密码登录，如图所示
-   ![pic](/images/012.png)
-   如果是用手机，**一定**！**一定**！**一定**要**请求桌面网站**以访问**电脑版**网页
-   如果不知道账号,但要求输入,尝试
```
root
admin
useradmin
网络SSID（就是WiFi名）
```
-   如果不知道密码，但要求输入，尝试下列等
```
password
123456
你的WiFi密码
```

#### 状态
![pic](/images/013.png)
-   概览：查看路由器共享指标及接入设备
-   防火墙：防止外界设备攻击
-   路由表：查看各个接口的工作负载和服务及接入设备
-   系统日志：仅限高级用户调试时查看（~~估计都是鸟语吧~~）
-   内核日志：同上述系统日志
-   系统进程：相当于任务管理器（**警告**，**请勿禁止系统关键进程**）
-   实时信息：监测路由器状态
-   WireGuard状态：WireGuard是一种现代VPN协议，特点是简单、快速、安全（~~翻墙使用~~）
-   负载均衡：调整路由器负载（**严重警告**，**请勿调整关键负载**，**严重会使路由器损坏**）

#### 系统
![pic](/images/014.png)
-   系统：调节路由器系统设置（**警告**，****请勿调整关键设置如时间**）
-   管理权：调节系统用户设置（**严重警告**，**请勿调整任何设置**，**严重会使路由器损坏**）
