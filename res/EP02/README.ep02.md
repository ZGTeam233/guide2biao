#### 开始之前
-   翻看路由器上的标签，一般在其背面/底面，找到管理地址/账号/密码，形如
```
品名：无线路由器
型号：RB03
输入：12伏1.5安
管理后台：http://192.168.31.1
管理后台账号：root
管理后台密码：password
```
-   用浏览器以*网址*形式访问管理地址，输入账号/密码登录，如图所示
-   ![pic](/images/012.png)
-   如果是用手机，**一定**！**一定**！**一定**要**请求桌面网站**以访问**电脑版**网页
-   如果不知道账号,但要求输入,尝试
```
root
admin
useradmin
网络SSID（就是WiFi名）
```
-   如果不知道密码，但要求输入，尝试
```
password
123456
你的WiFi密码
```
